# 📂 Assets (Recursos do Projeto VIGI)

Esta pasta contém os recursos visuais e documentais que suportam a arquitetura e a interface do **VIGI - Vigilante Financeiro**.

## 📑 Conteúdo Principal
- **`RoteiroLab.md`**: Memorial descritivo detalhando as etapas de desenvolvimento, arquitetura do agente e estratégias de segurança.
- **Diagramas**: Fluxogramas de interação entre o LLM (Gemini 2.5 Flash) e a base de dados (JSON/CSV).
- **Interface**: Mockups e prints da aplicação funcional desenvolvida em `ipywidgets` no Google Colab.

---
### 💡 Organização de Portfólio
Estes recursos são utilizados para alimentar a documentação principal do projeto e demonstrar a viabilidade técnica da solução proposta.
